namespace std : 표준 라이브러리 범위

std::cout, std::cin 처럼 쓰는 걸 모아놓은 공간

using namespace std; : std:: 안 붙이고 바로 cin, cout 쓰게 해줌

namespace (3~5줄 요약)

C++에서 이름 공간을 만드는 도구다.

같은 이름의 함수나 변수가 다른 곳과 겹치지 않게 구분해준다.

예: std라는 표준 라이브러리 공간 안에 cout, cin이 들어있다.

그래서 std::cout처럼 앞에 namespace 이름 붙여서 사용한다.

using namespace std;를 쓰면 앞에 std:: 안 붙여도 된다.
-----------------------------------------------------------------------------
fixed
    → 소수점 아래 기준으로 출력하겠다는 선언

setprecision(9)
    → 소수점 아래 9자리까지 보여줘라는 뜻

    cout << fixed << setprecision(9) << a / b;

    👉 “실수 결과를 소수 9자리까지 정확하게 출력해라”
    
    fixed 혼자 쓰면
    → “소수점 아래 기준으로 하겠다”는 모드만 전환
    → 몇 자리인지는 안 정함

setprecision(n)이
→ “소수점 아래 n자리까지 보여줘라”를 결정
    
fixed            // 기준을 바꿈
setprecision(9)  // 자릿수를 정함

한 줄 요약:
fixed = 기준 / setprecision = 자릿수

--------------------------------------------------------------------

#include <iomanip>
    → 출력 형식(포맷)을 다루는 도구 상자

    setprecision, fixed 같은 출력 조절 함수들이 들어 있음

    없으면 setprecision을 쓸 수가 없음

    👉 <iostream> = “출력하기”
    👉 <iomanip> = “어떻게 출력할지 꾸미기”

=====================================================================

입력값이 정수처럼 보여도 계산 결과가 소수면 실수 계산이 필요

int로 받으면 계산 순간에 이미 소수가 잘려버림

그래서 “계산을 실수로 하겠다”는 의도로 double로 받는 것

정수처럼 계산하면 → 정수 결과만 나옴
실수처럼 계산해야 → 실수 결과가 나옴
