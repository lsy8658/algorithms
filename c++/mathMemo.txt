원이 만나는지는 중심 사이 거리로 결정됨
그 거리를 반지름 합(r1+r2) 과 반지름 차(|r1−r2|) 와 비교
비교 결과로 교점 개수(0, 1, 2, -1)를 정함

중심이 같고 반지름도 같음 → 완전히 겹침 → -1

d가 너무 멀다 (d > r1+r2) → 안 만남 → 0

d가 딱 바깥에서 닿음 (d == r1+r2) → 1

d가 사이에 있음 → 두 점에서 만남 → 2

d가 딱 안에서 닿음 (d == |r1-r2|) → 1

d가 너무 가까움 (d < |r1-r2|) → 안 만남 → 0
