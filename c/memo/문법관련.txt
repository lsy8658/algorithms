자료형 배열이름[크기];

배열 선언 )
예) int arr[5]; → 정수 5개 저장 가능
초기화: int arr[3] = {1,2,3};


메모리 크기 )
sizeof(arr)는 배열 전체 바이트 크기를 반환함.
int가 4바이트라서 5 × 4 = 20이 출력됨.
값이 아니라 메모리 크기

배열 길이 )
int len = sizeof(arr) / sizeof(arr[0]);
for(int i = 0; i < len; i++) { /* 사용 */ }

- C 배열은 길이 정보 저장 안 함 → 컴파일 타임 크기만 앎.
그래서 보통 길이를 변수로 따로 관리함.
